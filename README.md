# ⚽ ScoreCast

تطبيق ويب لتوقع نتائج مباريات كرة القدم يوميًا والحصول على نقاط.  

## المميزات
- عرض مباريات اليوم (تجريبي أو حقيقي عبر API).  
- إدخال توقعات النتائج وحفظها.  
- واجهة جميلة باستخدام Tailwind CSS.  

## التشغيل
1. انسخ المشروع بالكامل.  
2. افتح ملف `index.html` في أي متصفح لمشاهدة النسخة التجريبية.  
3. لو عايز تعرض المباريات الحقيقية، ضيف `API_KEY` الخاص بك في الكود (`API_KEY` داخل الـ `<script>`).  

## GitHub Pages
1. أنشئ مستودع جديد.  
2. ضع الكود في المستودع مع `index.html`.  
3. افتح Settings → Pages → اختار الفرع `main` و `/ (root)`.  
4. اضغط Save وانتظر لينزل الموقع على الإنترنت.  

## ملاحظات
- النسخة التجريبية تعمل بدون API.  
- لو عايز ربط المباريات الحقيقية، سجل في [API-Football](https://api-football.com) وخد المفتاح.
